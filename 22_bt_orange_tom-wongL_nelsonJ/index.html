Ready to use bootstrap?

Make sure that you include the required meta tags!

<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">

Don't forget to link to the stylesheet for CSS to work!

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">


This is utilizing a content delivery network (CDN), specifically jsdelivr (free for open source). This CDN is providing all of the CSS associated with bootstrap. A separate link would be necessary for bootstrap's associated JavaScript (and any features that implement it will not work without this link).
